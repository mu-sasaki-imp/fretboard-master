# 🎸 FretboardMaster

ギターの指板上の音名を楽しく覚えるためのWebアプリです。

## ✨ 特徴

- **モード1: 音名を当てる** - 指定されたポジションの音名を回答
- **モード2: ポジションを当てる** - 指定された音名のポジションをクリック
- **モード3: 練習アシスト** - メトロノームと連動した実機練習サポート

## 🎮 ゲームモード

### モード1・2で選べる5つのゲームモード：

1. **🎯 フリープラクティス** - 時間制限なし、自分のペースで練習
2. **⚡ クイックファイア** - 10問を5秒以内に素早く回答
3. **⏱️ タイムアタック** - 60秒でできるだけ多く正解
4. **♾️ エンドレスチャレンジ** - 各5秒、ミスで終了。連続正解数を競う
5. **📚 ラーニングモード** - 20問をじっくり学習

## 📊 機能

- ✅ スコア記録（LocalStorage）
- ✅ ベストスコア表示
- ✅ 詳細な結果画面（正解率、平均回答時間、間違えた問題のリスト）
- ✅ 練習時間の記録（モード3）
- ✅ 🎵 **音再生機能** - 出題された音がギター音またはサイン波で鳴る（オン・オフ切替可）
- ✅ 🎼 **スケールフィルタ** - キーとスケール（Major/Natural Minor）を指定して出題範囲を限定
- ✅ レスポンシブデザイン

## 🚀 使い方

### オンラインで使う
デモサイト：[https://mu-sasaki-imp.github.io/fretboard-master/](https://mu-sasaki-imp.github.io/fretboard-master/)

### ローカルで動かす

```bash
# 依存関係のインストール
npm install

# 開発サーバーを起動
npm run dev

# ビルド
npm run build

# ビルド結果をプレビュー
npm run preview
```

## 🛠️ 技術スタック

- **Vue.js 3** - リアクティブなUIフレームワーク
- **Vite** - 高速ビルドツール
- **Web Audio API** - メトロノーム音の生成
- **LocalStorage** - スコアの永続化
